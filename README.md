### 1. Persiapan:
- Install dependensi:
  ```bash
  pip3 install telethon python-dotenv
  ```
- Buat file `.env`:
  ```bash
  nano .env
  ```
  ```plaintext
  API_ID=YOUR_API_ID
  API_HASH=YOUR_API_HASH
  ```
- Buat folder:
  ```bash
  mkdir sessions webview_results query
  ```

### 2. Menjalankan Skrip:
- Jalankan skrip:
  ```bash
  python3 main.py
  ```
  ```bash
  python main.py
  ```
